# Chrome Github Reminder
- できるリスト

- [x] GithubのTokenの認証
- [x] Pull Requestのページなら、保存できる
- [x] 保存したPRに対して定期的にアップデートチェックを行う <= todo
- [x] アップデートのあるページは、popup上で表示される
- [x] popup上で保存したPRの行をクリックすることで、次のアップデートまでその行が消える。また、新しいタブでそのページが開く
- [] マージされれば、保存リストから削除される
